## 구현된 게임 모드

### 전략 모드 (7개)
1. **Standard (클래식 바둑)** - 일반적인 클래식 바둑
2. **Capture (따내기 바둑)** - 정해진 개수의 돌을 먼저 따내는 사람이 승리
3. **Speed (스피드 바둑)** - 피셔 방식 시간 제한
4. **Base (베이스 바둑)** - 비밀 베이스 돌 배치
5. **Hidden (히든 바둑)** - 상대에게 보이지 않는 히든 돌
6. **Missile (미사일 바둑)** - 미사일로 돌 이동
7. **Mix (믹스룰 바둑)** - 여러 규칙 조합

### 재미 모드 (6개)
8. **Dice (주사위 바둑)** - 주사위 결과만큼 돌 놓기
9. **Omok (오목)** - 5개 연속 승리
10. **Ttamok (따목)** - 오목 변형 (따내기 포함)
11. **Thief (도둑과 경찰)** - 역할 기반 추적 게임
12. **Alkkagi (알까기)** - 돌 튕기기 게임
13. **Curling (바둑 컬링)** - 목표 지점에 가까이 놓기

## 기술 스택

### Backend
- **Fastify** - 고성능 HTTP 서버
- **tRPC** - End-to-end 타입 안전 API
- **Prisma** - 데이터베이스 ORM
- **PostgreSQL** - 데이터베이스
- **WebSocket (ws)** - 실시간 통신
- **Pino** - 구조화된 로깅
- **Zod** - 런타임 검증

### Frontend
- **Next.js 14+** - React 프레임워크 (App Router)
- **React 19** - UI 라이브러리
- **TypeScript** - 타입 안전성
- **Zustand** - 상태 관리
- **Tailwind CSS** - 스타일링
- **tRPC Client** - 타입 안전 API 클라이언트

### 개발 도구
- **Vitest** - 테스트 프레임워크
- **Turbo** - 빌드 시스템
- **ESLint** - 코드 린팅
- **Prettier** - 코드 포맷팅
- **GitHub Actions** - CI/CD

### 배포 전 확인 사항
1. 환경 변수 설정
   - `DATABASE_URL` - 프로덕션 데이터베이스 URL
   - `JWT_SECRET` - 최소 32자 이상의 시크릿 키
   - `NODE_ENV=production`
   - `PORT` - 서버 포트 (기본: 4000)
   - `ALLOWED_ORIGINS` - CORS 설정

2. 데이터베이스
   - 프로덕션 데이터베이스 생성
   - Prisma 마이그레이션 실행
   - 데이터베이스 백업 설정

3. 테스트
   - 모든 테스트 통과 확인
